Deploy your agent to TikTok through the "Deployments" section in Agent settings.

## Setup Process

TikTok Setup is a straightforward OAuth connection. Simply click 'Connect TikTok Account', then follow the instructions.

## Using the TikTok Tools

The following tools are available for your agent to use once it is deployed to TikTok:
- `tiktok_post` - Post a video to a TikTok
