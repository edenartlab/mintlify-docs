The easiest way to make creations with Eden is through the [creation tool frontend](https://app.eden.art/create).

<Tip>
You can also interact with the creator tool [through the SDK](/guides/build).
</Tip>

## Overview

Eden offers a number of generative pipelines for making images and videos, and we are always adding the latest and greatest models. The pipelines are divided into a number of tools which are optimized for different visual and audio tasks.

## Tools

To see the full available tools, see the [Tools](https://app.eden.art/tools) page in the app, or check out the [Tools Reference](/tools/flux_kontext) in this documentation.
